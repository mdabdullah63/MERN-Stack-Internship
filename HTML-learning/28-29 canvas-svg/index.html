concepts of canvas-svg

canvas and svg are two different technologies for drawing graphics on the web.

canvas

canvas is a bitmap-based drawing API that allows you to draw graphics using JavaScript. it is a low-level API that provides a lot of control over the drawing process, but it can be more difficult to use than svg.

svg

svg (scalable vector graphics) is a vector-based drawing format that allows you to create graphics using XML. it is a high-level API that is easier to use than canvas, but it may not provide as much control over the drawing process.
